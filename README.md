# Machine Learning with Python Projects Collection

This repository contains a collection of machine learning and artificial intelligence projects developed using Python, TensorFlow, and deep learning techniques.
The projects focus on different AI domains including computer vision, natural language processing, regression models, game AI, and classification systems.

The repository was created as part of practical machine learning training and project-based learning.

---

# Projects Included

## 1. Cat and Dog Image Classification

A deep learning computer vision project that classifies images of cats and dogs using Convolutional Neural Networks (CNNs).
The model learns visual patterns from image datasets and predicts the correct animal category.

Main concepts:

* Image classification
* CNNs
* Computer Vision
* TensorFlow

File:

* `fcc_cat_dog.ipynb`

---

## 2. Health Cost Prediction using Regression

A machine learning regression project that predicts medical insurance costs based on patient information such as age, BMI, smoking habits, and other health-related features.

Main concepts:

* Regression models
* Data preprocessing
* Prediction systems
* Machine learning workflows

File:

* `fcc_predict_health_costs_with_regression.ipynb`

---

## 3. SMS Spam Classification using Deep Learning

A Natural Language Processing project that classifies SMS messages as spam or ham using a Bidirectional LSTM neural network built with TensorFlow.

Main concepts:

* NLP
* Text classification
* LSTM networks
* Deep learning

File:

* `fcc_sms_text_classification.ipynb`



---

## 4. Rock Paper Scissors AI Bot

An AI-based Rock Paper Scissors bot that learns strategies to defeat different opponents by analyzing previous moves and adapting its gameplay.

Main concepts:

* Game AI
* Strategy prediction
* Pattern analysis
* Python logic systems

Files:

* `RPS_game.py`
* `main.py`
* `test_module.py`



---

## 5. Machine Learning Practice and Testing Utilities

Additional helper scripts and testing modules used for validating models, running experiments, and testing AI logic implementations.

Files:

* `test_module.py`
* `main.py`
